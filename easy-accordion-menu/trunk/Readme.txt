== Description ==

Плагин Easy Accordion Menu позволяет организовать на вашем сайте простое и элегантное аккордеон меню (раскрывающееся меню).

== Installation ==
	
1. Загрузите плагин "Easy Accordion Menu" в директорию `/wp-content/plugins/`;
2. Активируйте "Easy Accordion Menu" на странице `Плагины` в панели управления.
