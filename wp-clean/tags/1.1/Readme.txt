== Description ==

Plugin WP Clean removes meta-tags and disables unnecessary functions in CMS WordPress.

WP Clean removes:

* version of wordpress;
* meta-tag to work with the blog client Windows Live Writer.
* links to neighboring articles;
* dns-prefetch;
* extra links from admin bar;
* deletes the entries in the admin panel that you entered the password/username is not correct;
* static page from the search;
* wp-json;
* oembed;
* embed.

WP Clean disable:

* RSS feed;
* trackback (on your website);
* Emoji WordPress (smiles Emoji).

function temporarily disabled: standard widgets (archives, calendar, categories, meta, pages, recent comments, recent entries, rss, search, and tag cloud);

P.S: If you have any suggestions on how to improve the plugin, please send email to <cite>info[@]artemsannikov.ru</cite>

== Installation ==
	
1. Upload plugin "WP Clean" to the `/wp-content/plugins/` directory;
2. Activate the plugin "WP Clean" through the 'Plugins' menu in WordPress.
