== Description ==

Плагин Disable Images Compress позволяет отключить сжатие изображений при загрузке на сайт.

P.S: Если у вас остались какие-либо вопросы пишите на почту <cite>info[@]artemsannikov.ru</cite>.

== Installation ==
	
1. Загрузите плагин "Disable Images Compress" в директорию `/wp-content/plugins/`;
2. Активируйте "Disable Images Compress" на странице `Плагины` в панели управления.

== Frequently asked questions ==
